# Obsidian Tutorial


### Link a block to other page
```
![[page_name#title_name]]
```

### Different Blockquotes Type

| 語法                                           | 顏色  |
| -------------------------------------------- | --- |
| `[!note]` / `[!info]`                        | 藍   |
| `[!tip]` / `[!hint]` / `[!important]`        | 綠   |
| `[!warning]` / `[!caution]` / `[!attention]` | 橘   |
| `[!danger]` / `[!error]`                     | 紅   |
| `[!success]` / `[!check]` / `[!done]`        | 綠   |
| `[!question]` / `[!help]` / `[!faq]`         | 黃綠  |
| `[!failure]` / `[!fail]` / `[!missing]`      | 紅   |
| `[!bug]`                                     | 紅   |
| `[!example]`                                 | 紫   |
| `[!quote]` / `[!cite]`                       | 灰   |
| `[!abstract]` / `[!summary]` / `[!tldr]`     | 青   |
