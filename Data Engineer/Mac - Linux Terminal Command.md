# Mac / Linux Terminal Command

tags: #terminal #tools #linux #mac
source: [[Isaac's Note]]

---

### Print Working Directory

```shell
pwd
```
>> /Users/wen

### List

```shell
ls        # 列出所有目前資料夾下的東西
ls -lh    # 列出所有東西的詳細資訊
```

### Change Directory

```shell
cd Music  # 進到 Music 這個資料夾裡
cd ..     # 回到上一個資料夾
cd ~      # 回到家目錄
```

### Tab 自動補完

```shell
cd Mu [Tab]   # 會自動把後面的字補完
```

### Make / Remove Directory

```shell
mkdir Project  # 新增資料夾
rmdir Project  # 移除資料夾
```

### Copy File

```shell
cp [要複製的檔案] [複製到哪裡]
cp Airflow Project
```

### Move

```shell
mv Airflow Project
```

### Remove

```shell
rm Project
```

### Clear

```shell
clear  # 清除所有文字
```

### Calendar

```shell
cal
cal 2022
cal 09 1996
```
