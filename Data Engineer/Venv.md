## 建立 Python 虛擬環境

### cd to the folder

```bash
cd /path/to/your/project
```

### Create Venv in the folder

```bash
python3 -m venv <venv name>
```

### Activate the venv

```bash
source venv/bin/activate

# 範例
source ~/argilla/argilla_env/bin/activate
```
